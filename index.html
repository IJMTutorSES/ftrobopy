ftrobopy

Das (import-) Modul kann unter Betriebssystemen genutzt werden, die Python (Version 2.7 oder 3.x) und POSIX Sockets unterstuetzen. Dazu gehoeren z.B. Linux und Mac OS X Systeme aber auch Windows 7 (mit einer entsprechenden Python Installation).

ftrobopy.py           # Dies ist das Modul
manual.pdf            # Eine Beschreibung aller Klassen und Methoden
SimpleDemoExplorer.py # Eine kleine Demo Anwendung

Entweder wird die Datei ftrobopy.py in das site-packages Verzeichnis der Python Installation des Systems kopiert, damit sie automatisch gefunden wird oder man kopiert die Datei in das gleiche Verzeichnis in dem man auch seine Python-Programme zur Steuerung des TXT abgespeichert hat.
